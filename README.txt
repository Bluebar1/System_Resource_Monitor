# MyTaskManager
Informatics Final Project  
 
## First Time Setup
Make sure python version <= 3.9.x  
cd into directory in windows command prompt 
Running .bat file to install virtualenv and activate virt 
> first_time  
Install dependencies  
> pip install -r requirements.txt  

## Every time:
> virt\Scripts\activate  
> python kivy_app.py    

Running the main.py instead will update the assets/stats.json file.  


# Project Overview
My Goal is to create a better version of the windows 10 task manager.
